Autotabs with sliding doors

jquery.autotabs is ready for sliding doors out of the box. There is no need for any javascript configuration. Simply add your own sprite and your stylesheet.

            background: transparent url("images/btn_blue_sprite.gif") no-repeat right -140px;
            border: none;
            padding: 0 20px 14px 0;
          }

          .sliding_doors ul.autotabs li.autotab a {
            background: transparent url("images/btn_blue_sprite.gif") no-repeat left top;
            padding: 5px 0 3px 20px;
            display: block;
            color: #fff;
          }

          .sliding_doors ul.autotabs li.autotab.current,
          .sliding_doors ul.autotabs li.autotab:hover {
            background-position: right -210px;
          }

          .sliding_doors ul.autotabs li.autotab.current a,
          .sliding_doors ul.autotabs li.autotab:hover a {
            background-position: left -70px;
          }
